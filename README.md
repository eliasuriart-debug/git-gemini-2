# Mi proyecto de practica

¡Bienvenido! Este es un repositorio creado desde cero para dominar los fundamentos de **Git** y **GitHub** y la gestión de archivos esenciales como `.gitignore`.

## Tecnologías utilizadas
* **HTML5** para la estructura del sitio.
* **Git** como sistema de control de versiones.
* **GitHub** para el alojamiento del código en la nube.

## Aprendizaje clave
1. Inicialización de repositorios desde el absoluto cero.
2. Manejo de las 4 zonas de Git (Working Directory, Staging Area, Local Repo y Github).
3. Configuración y codificación correcta del archivo `.gitignore`.
